Day 2: ES6 Practice Set

Template Literals & Destructuring

How to Use This File

  1. Open this `index.html` file in your browser to see the problems.
  2. Open the `practice.js` file in your code editor.
  3. Write your solutions in `practice.js` and check the browser console for the output.

Part 1: Template Literals Practice

Problem 1.1: Basic Greeting
Create a string that says "Hello, my name is Alex and I am 28 years old." using the two variables provided.

Problem 1.2: Simple Calculation
Create a string that says "The total cost is $125." by calculating `price * quantity` inside the template literal.

Problem 1.3: Multi-line String
Create a multi-line string that represents an address, with each part on a new line.

Part 2: Destructuring Practice

Problem 2.1: Basic Object Destructuring
Extract the `title` and `author` from the `book` object into their own `const` variables.

Problem 2.2: Destructuring with Renaming
Extract the `id` from the `user` object, but rename the new variable to `userID`.

Problem 2.3: Nested Destructuring
Extract the `street` and `city` from the `person` object's nested `address` object.

Part 3: Combined Challenge

Your task is to create a function called `createEventAnnouncement`. This function should accept an `event` object as its parameter. Inside the function, use **destructuring** to unpack the object's properties and a **template literal** to create and return a formatted HTML string for an event announcement card. Finally, print the result of this function to the console.
